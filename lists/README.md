# Lists
* data structures that group sequences of elements
* can have elements of several types
* can also mix different types within the same list

### Accessing elements of List
```
list1 = [1, 2, 3, 4, 5, 6]
list2 = [8, 9]

print(list2)
8
9

print(list1 + list2)
1
2
3
4
5
6
8
9

print(list1[0:4])
1
2
3
4
```
